New Rewrite Rule

PURLs are used to curate the URL resolution process, thus solving the problem of transitory URIs in location-based URI schemes like HTTP.

Partial PURLs are a special class of PURL that must end with a '/' and allow a hierarchy of URLs to be mapped to another resource. The request pattern may include regular expressions in the path subsequent to the PURL name.

Other URLs cannot end with a '/' and the request pattern only applies to the query string.
