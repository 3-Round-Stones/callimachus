• Query the database using the SPARQL language.
  • Namespace prefixes are optional and if omitted the default SPARQL Profile will be used.
  • Additional RDF graphs can be uploaded using the RDF Graph Interface.
  • To get started try listing some of the different types available in the database. 
    SELECT DISTINCT ?type
WHERE {
  ?thing a ?type
}
ORDER BY ?type
LIMIT 50
