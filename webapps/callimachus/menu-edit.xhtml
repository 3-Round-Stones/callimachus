• To replace the icon or logo, click the × button, then the chevron to open a file upload dialogue
  • The icon should be of type PNG, JPEG, GIF, or ICO and should be 16×16 or 32×32 pixels in size
  • The logo should be of type PNG, JPEG, or GIF and should be 128x128 pixels in size
  • Click on the menu entry you want to edit
  • Use ENTER key to create a new entry at the start or end of a line
  • Use TAB key to indent, or shift-TAB to outdent
  • Use SPACE to separate a label from a full URL
  • Drag links or text to an empty list item to insert, then click outside of the edit area
