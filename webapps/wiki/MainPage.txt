= Welcome to Callimachus

If you're seeing this page via a web browser, it means you've installed Callimachus successfully. Congratulations!

To finish the setup process visit [[/group/admin?usersetup|The User Setup Page]] and choose an administrator password. This page is only accessable from the local machine and only until an administrator password is choosen.

As you may have guessed by now, this is the default Callimachus home page. It can be found on the local filesystem at:

webapps/wiki/MainPage.txt

In the root of the Callimachus installation directory. If you're seeing this page, and you don't think you should be, then you're either a user who has arrived at new installation of Callimachus, or you're an administrator who hasn't got his/her setup quite right. Providing the latter is the case, please refer to the [[http://callimachusproject.org/|Callimachus Website]] or the README.txt file provided in the install directory.

You can download sample applications following the [[http://code.google.com/p/callimachus/wiki/GettingStarted|Getting Started page]].

Callimachus mailing lists are available at the [[http://callimachusproject.org/|Callimachus Website]].

Thanks for using Callimachus!
