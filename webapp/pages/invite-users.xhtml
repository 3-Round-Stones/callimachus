Invite Users Group permissions

Groups provide a way to assign permissions to a named set of users. User permissions through groups can help generalize resource's permissions.

Users Invite

Members of this group can view, discuss, document, link, and upload binary resources.

Admin Invite

Members of this group have the ability to edit other user accounts and access to modify the underlying data store.

Power Invite

Members of this group can access all data in the underlying data store.

Staff Invite

Members of this group can design websites and develop applications.
