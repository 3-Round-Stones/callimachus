Welcome to Callimachus

If you're seeing this page via a web browser, it means you've setup Callimachus successfully. Congratulations!

As you may have guessed by now, this is the default Callimachus home page. It can be found on the local filesystem at:

webapps/index.html

In the root of the Callimachus installation directory. If you're seeing this page, and you don't think you should be, then you're either a user who has arrived at new installation of Callimachus, or you're an administrator who hasn't got his/her setup quite right. Providing the latter is the case, please refer to the Callimachus Website or the README.txt file provided in the install directory.

You can download sample applications from the Callimachus Website.

Callimachus mailing lists are available at the Callimachus Website.

Thanks for using Callimachus!
